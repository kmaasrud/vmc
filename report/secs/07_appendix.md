# Appendix

## Hermite polynomials

The Hermite polynomials are given by the following sequence

$$ H_n(x) = c_nx^n + ... + c_0x^0 $$

where the coefficients $c_n$ are given by the sequence shown below [@Jovovic2001]:

$$c_i = 1, 0, 2, -2, 0, 4, 0, -12, 0, 8, 12, 0, -48, ...$$
